# README

Web application that allows user to play rock-paper-scissors.

# Configuration
1. Clone repository
```bash
git clone git@github.com:tobiasz-p/rock-paper-scissors.git
```

2. Install gems
```bash
cd rock-paper-scissors
bundle install
```

3. Run rails server
```bash
rails s
```
4. Open browser and go to http://localhost:3000
5. Have fun!
